City Defenders

Dec 2019 - Jan 2021

Project Group Size: 2

Development Time: 25 days

It's a university project for my VR class. We had decided to make a game for the Gear VR since that was the headset we had access to. I was responsible for the programming, modelling and almost all of the game design. Since the game was for a mobile device running VR, I had to really limit the art potential. All of the models are textureless and I've put a limit to particle spawn rate. I've also wrote some special shaders for various elements. It was really interesting to work on VR because it differs so much from PC. I've enjoyed designing a completely new 3D UI and the way player gets information they want. The feel of VR is unique and the gameplay elements has to fit in to that feeling. I've tried to add everything organic. For example, I could've just teleported the enemies but I wanted to support immersion so I've made alien drop ships with simple AI. Since Gear VR is no longer supported, I couldn't put the game on oculus store but you can install the .apk yourself. I want to make a game for a real vr headset someday

Highlight Of The Project

VR wheel tool selection UI. Mobile-VR optimizations. Wave Function Collapse city generation
